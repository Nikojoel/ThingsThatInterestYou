    Search for thousands of different events happening in Helsinki metropolitan area. TTIY uses Helsinki Linked Events API to gather information about the events and OpenStreetMap with Leaflet JavaScript library to display the locations of the specific events.

    Filter your searches by specifing your desired starting aswell as ending date and keyword eq. music. Results are independent of your current location. After the search is complete, scroll down to load more events to choose from. Clicking a specific event will open up a new tab.

    Created as a web application project in association with Metropolia University of Applied Sciences and only non-marketed and open source APIs are used. Check out the source code from GitHub. Creators are Niko Holopainen, Jetro Saarti and Daniel Eggleton.
