Things That Interest You

Search for thousands of different events happening in Helsinki metropolitan area. TTIY uses Helsinki Linked Events API to gather information about the events and OpenStreetMap with Leaflet JavaScript library to display the locations of the specific events.

Filter your searches by specifing your desired starting aswell as ending date and keyword eq. music. Results are independent of your current location. Load up to 20 events per one search to choose from.

Created as a web application project in Metropolia University of Applied Sciences and only non-marketed and open source APIs are used. Creators are Niko Holopainen, Jetro Saarti and Daniel Eggleton.
